# Azura Guide v2

Static luxury editorial redesign of the Azura menu guide.

## Run
Open `index.html` in a browser, or deploy the folder to any static hosting.

## What changed
- Full visual redesign toward Mediterranean luxury / editorial wine guide.
- Warm paper background, serif typography, premium cards, bottle-first wine/bar cards.
- Full bottle images are used for wine and bar instead of label crops.
- Gallery-style dark lightbox.
- Mobile-first layout.
